# 🎮 Text Adventure Game in C

## 📌 Description

This is a simple console-based text adventure game written in C.
The player explores different locations like a forest and a cave, encounters random events, and tries to survive while increasing their chances of winning.

---

## 🕹️ How to Play

* You start with **100 health**
* Choose actions from the menu:

  * **1 → Forest**: Random events (find sword, fight animal, gain health)
  * **2 → Cave**: Fight monster (win only if you have a sword)
  * **3 → Rest**: Recover health
  * **4 → Exit**: Quit the game
* Your goal is to **find a sword and defeat the monster in the cave**

---

## ⚙️ Features

* Random events using `rand()`
* Health system ❤️
* Simple combat mechanics ⚔️
* Win/Lose conditions 🎯

---

## 🧑‍💻 Requirements

* C Compiler (GCC recommended)

---

## ▶️ How to Run

1. Save the code as:

   ```
   game.c
   ```

2. Compile the program:

   ```
   gcc game.c -o game
   ```

3. Run the executable:

   ```
   ./game
   ```
